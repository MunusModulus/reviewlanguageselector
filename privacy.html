Review Language Selector Privacy Policy

Review Language Selector is a Chrome extension that helps users filter Google reviews by selected language and view language-filtered review scores.

Data Collection

This extension does not collect, sell, or share personal information.

Review Data

The extension reads review text and rating information displayed on the current page only to filter reviews and calculate the displayed average rating. This processing is performed locally in the user's browser.

Stored Settings

The extension may store user settings such as selected language, maximum review count, and display preference using Chrome's extension storage. These settings are used only to keep the extension's behavior consistent.

External Transmission

The extension does not transmit review contents, ratings, browsing history, or user settings to an external server.

Third-Party Services

This extension works on Google review pages. Google services are governed by Google's own terms and privacy policies.

Contact

For support or questions, please use the GitHub repository for this project.
