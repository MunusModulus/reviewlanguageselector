# Review Language Selector

Review Language Selector is a Chrome extension that helps you filter Google reviews by selected language and check language-filtered review scores more easily.

Currently, only Japanese review filtering is supported.

## Features

- Filter Google reviews by selected language
- Show the average rating of filtered reviews
- Show the number of filtered reviews
- Choose how review score results are displayed
- Set the maximum number of reviews to check
- Re-check newly loaded reviews after scrolling when possible

## Current language support

Currently, only Japanese reviews are supported.

Support for additional languages may be added in future updates.

## Notes

Language detection may not always be accurate.

This extension works on Google review pages, but the layout and behavior of Google services may change over time.
